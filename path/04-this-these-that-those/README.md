# THIS, THAT, THESE, THOSE

[← Regresar a notas](../../README.md) <br>

---

Los pronombres demostrativos `this`, `that`, `these` y `those` se usan para señalar objetos, personas o situaciones dependiendo de su proximidad en relación con el hablante.

## 1. This (este, esta, esto)
Se usa `this` para referirse a algo `singular` que está `cerca` del hablante.

> - `This book is interesting.` : <u>Este</u> libro es interesante.
> - `Is this your pen?` : ¿Es <u>este</u> tu bolígrafo?
> - `I like this song.` : Me gusta <u>esta</u> canción.

---

## 2. That (ese, esa, eso, aquel, aquella, aquello)
Se usa `that` para referirse a algo `singular` que está `lejos` del hablante.

> - `That house is big.` : <u>Aquella</u> casa es grande.
> - `Who is that person?` : ¿Quién es <u>esa</u> persona?
> - `I don’t like that idea.` : No me gusta <u>esa</u> idea.

---

## 3. These (estos, estas)
Se usa `these` para referirse a algo `plural` que está `cerca` del hablante.

> - `These apples are fresh.` : <u>Estas</u> manzanas están frescas.
> - `Are these your keys?` : ¿Son <u>estas</u> tus llaves?
> - `I bought these shoes yesterday.` : Compré <u>estos</u> zapatos ayer.

---

## 4. Those (esos, esas, aquellos, aquellas)
Se usa `those` para referirse a algo `plural` que está `lejos` del hablante.

> - `Those mountains are beautiful.` : <u>Aquellas</u> montañas son hermosas.
> - `I don’t know those people.` : No conozco a <u>esas</u> personas.
> - `Those chairs look old.` : <u>Esas</u> sillas parecen antiguas.
